# Python While 


## Introdução

Entenda como funcionam os loops WHILE em Python para repetir blocos de código com base em uma condição.

## Exemplos Práticos

*   **Contador Simples:** Contagem de 0 a 10.
*   **Contagem Regressiva:** Contagem de 10 a 1.
*   **Soma de Números:** Soma números digitados pelo usuário até que -1 seja inserido.
*   **Validação de Senha:** Pede uma senha até que a correta seja informada.

## Exercícios

*   **Questão 01:** Contagem regressiva de 10 a 1 com mensagem final.
*   **Questão 02:** Exibe números pares de 0 a 20.
*   **Questão 03:** Soma números digitados até que -5 seja inserido.
*   **Questão 04:** Pede um código para abrir um cofre.
*   **Questão 05:** Contagem regressiva de 20, diminuindo de 2 em 2.
